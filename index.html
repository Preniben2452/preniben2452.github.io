Download the Latest Version

This program provides a friendly interface to operate your Maslow CNC machine. It handles communication with your hardware using FluidNC, guides you through the calibration process, and manages the state of your machine during cuts.

📋 What this software does

Maslow-desktop acts as the bridge between your computer and your machine. It translates your design files into commands the machine understands. The software simplifies setup, monitors your progress, and prevents common errors during operation.

  • Calibration Wizard: Step-by-step instructions ensure your machine is accurate and ready for projects.
  • State Management: The system tracks your machine status to prevent accidental errors while it moves.
  • G-Code Sending: Easily load and run design files from your computer to the cutting tool.
  • Visual Feedback: See where your cutter is on your workspace at all times.

📥 How to install the program

Follow these steps to set up the software on your Windows computer.

  1. Visit the official releases page to access the installer file.
  2. Look for the file ending in .exe under the assets section of the latest release.
  3. Click the file to download it to your Windows folder.
  4. Open the downloaded file. Your computer might show a security prompt because the app is new. Click "More Info" and then "Run anyway" to proceed.
  5. Follow the instructions on the screen to finish the installation.
  6. Launch the maslow-desktop icon from your desktop or start menu to begin using your machine.

💻 System requirements

The software requires an active connection to your Maslow CNC machine via a USB cable. We recommend using a computer running Windows 10 or Windows 11. Ensure your machine has the latest FluidNC firmware installed before you connect for the first time.

🛠️ Using the calibration wizard

The first time you open the program, the wizard will launch. Do not skip this step. The wizard measures the physical dimensions of your workspace. It asks you to move the machine to specific points and confirm the position. This data allows the software to calculate the correct math for your cuts.

🔍 Troubleshooting common issues

If the program does not see your machine, check that the USB cable is tight on both ends. Ensure no other applications are using the same communication port. If the software freezes, close the application, cycle the power on your Maslow CNC, and restart the software.

📝 About the project

This software exists to provide a modern experience for Maslow users. It is built using Rust and Svelte, which ensures a fast and reliable interface. It focuses on stability and clarity for woodworkers who want to focus on their projects rather than struggling with software settings.
